Include images in your wordpress RSS.

== Description ==

Simple plugin that allows to include the first attached image of your post in your wordpress rss, rss2.
You can select the  size of the images on the admin panel.

== Installation ==

1. Upload ‘wp rss images’ folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings/WP RSS Images and select the image size (Thumbnail, Medium, Full) and feed type (rss, rss2).
